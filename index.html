Abstract

For our project, we attempted to automatically label different sounds from the environment with the help of non-negative matrix factorization.

Project Description

First, we will more specifically describe the data set we used to test and train our model. As described above, we pulled players’ single season data from www.basketball-reference.com. We then paired the 3,823 instances we obtained with the corresponding NBA 2K rating. For each instance, we had 26 attributes. Those attributes are first name, last name, player’s age, the player’s team, and then various season totals, such as total points scored and total number of two point shots taken.
